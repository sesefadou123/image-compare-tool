# 图片对比小工具

一个纯前端小工具，用来对比两组图片的：

- `MD5` 是否一致
- `aHash`
- `dHash`
- `pHash`

支持：

- 整个文件夹上传对比
- 单张图片上传对比
- 多张图片上传对比

## 使用方式

直接打开 [index.html](./index.html) 即可使用。

如果你希望通过本地服务打开，也可以在当前目录执行：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080/image-compare-tool/`。

## 配对规则

默认使用自动配对：

1. 优先按目录的相对路径配对
2. 如果没有相对路径重合，则按文件名配对
3. 仍然无法匹配时，按上传顺序配对

## 结论规则

- `aHash / dHash / pHash` 任意差值 `<= 5` 时，该单元格会标红
- 结论颜色：
  - `相似`：红色
  - `一般`：黄色
  - `不相似`：绿色

当前页面中的“相似 / 一般 / 不相似”是基于三种感知哈希的综合判断，适合作为快速人工筛查辅助，不建议替代专业图像识别流程。
